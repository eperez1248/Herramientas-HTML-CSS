Pizzas

Pizza vegetal: Para vegetarianos o simplemente para los que queráis hacer una comida liguera, os ofrecemos esta pizza vegetal a base de tomates cherry, mozzarella, aceitunas negras y cebolla, con hojas de albahaca por encima.

Pizza de carne picada: Para los carnívoros aquí tenéis esta jugosa pizza a base de carne picada 100 % vacuno y pimientos verdes, que os dará un aporte extra de proteínas.

Pizza hawaiana: Para aquellos a los que os gusta mezclar sabores, aquí tenéis nuestra exquisita pizza hawaiana, con mezcla de prosciutto italiano, piña y con una base de salsa de tomate casero… ¡¡Extrajugosa!!
